# 筐体設計要件仕様書：XIAO ESP32S3 Sense 音声記録デバイス

## 1. プロジェクト概要
Seeed Studio XIAO ESP32S3 Senseを使用し、マイク録音・Wi-Fi送信を行うウェアラブルデバイスの筐体設計要件をまとめる。
本デバイスはリポバッテリー駆動であり、独自の電源回路（理想ダイオード+スイッチ）を内蔵する。

## 2. 構成部品
* **メインボード:** Seeed Studio XIAO ESP32S3 Sense (W:21 x D:17.5 x H:15 mm ※概算、カメラ含む)
* **バッテリー:** LiPo 1Cell (3.7V) 300~600mAh
* **電源制御:** スライドスイッチ + 理想ダイオード(XC8111)実装基板
* **配線:** 耐熱電子ワイヤー

---

## 3. 設計・実装ガイドライン

### 3.1 静電気対策 (ESD Protection)
冬場の静電気によるIC破壊および誤動作を防ぐための設計を行う。
* **端子のオフセット配置:**
  * USB Type-CポートおよびmicroSDスロットは、筐体表面から **1.0mm ～ 1.5mm 奥まらせて** 配置する（指先が金属端子に直接触れるのを防ぐため）。
  * ケーブル接続用に、コネクタ周辺の開口部は十分なクリアランスを確保すること。
* **スイッチの絶縁:**
  * 金属製ノブのスライドスイッチを使用する場合、**樹脂製のキャップ**を被せるか、スイッチ自体を内部に配置し樹脂製のアームで操作する構造とする。
* **絶縁距離の確保:**
  * 内部配線のはんだ付け箇所が、筐体外部（または筐体の隙間）から直接触れられない構造にする。

### 3.2 熱対策 (Thermal Management)
ESP32S3の高負荷時の発熱（50℃以上）と、リポバッテリーの熱保護を考慮する。
* **通気口（ベント）の設置:**
  * XIAOメインボード（特に銀色のRFシールド缶付近）の直上または側面に、放熱用の**スリットまたは穴**を設ける。
  * 可能であれば、空気の対流が起きるよう吸気・排気の2箇所に通気口を設ける。
* **断熱・隔離配置:**
  * **「CPUの発熱面」と「バッテリー」を直接密着させない。**
  * 空間的余裕（エアギャップ 2mm以上）を設けるか、間に断熱素材（スポンジ等）を挟めるスペースを確保する。

### 3.3 機構・配置設計 (Mechanical Design)
* **マイク用開口部:**
  * Senseボード上のマイク位置に合わせて、直径 **1.5mm ～ 2.0mm** の穴を開ける。
  * 音響特性確保のため、基板と筐体のマイク穴の間に隙間ができないよう設計する（エプトシーラー等で埋めるとなお良し）。
* **SDカードへのアクセス:**
  * 頻繁な抜き差しを想定し、**「爪やピンセットが入る切り欠き」** をスロット周辺に設ける。
  * スロット開口部が狭すぎると、プッシュ-プッシュ機構で飛び出したカードが筐体に引っかかり取り出せなくなるため注意。
* **カメラフレキの保護:**
  * カメラモジュールのFPC（オレンジ色の薄いケーブル）には、曲げストレスや挟み込みの負荷をかけないこと。
  * 筐体組立時にケーブルを噛み込まないための「逃げ（スペース）」を作る。

### 3.4 安全対策・バッテリー格納 (Safety & Battery)
* **バッテリー膨張マージン:**
  * リポバッテリーの格納スペースは、バッテリー厚みに対して **+0.5mm ～ 1.0mm** の余裕を持たせる（経年劣化や発熱によるわずかな膨張を許容するため）。
* **配線ガイド:**
  * 手はんだによる配線（スイッチ、理想ダイオード基板含む）が内部で暴れないよう、ケーブルガイドや固定用の突起を設ける。
  * 筐体を閉じる際、配線が挟まるリスクのある箇所を排除する。

## 4. 設計チェックリスト (Design Checklist)

- [ ] **USBポート周辺:** コネクタは奥まっているか？ 太めのケーブルでも挿さる開口径か？
- [ ] **SDカード:** カードのプッシュ排出時に指が入るスペースはあるか？
- [ ] **マイク穴:** 基板のマイク実装位置と筐体の穴はずれていないか？
- [ ] **排熱:** CPU付近に通気口はあるか？
- [ ] **バッテリー:** CPUの熱が直撃しない配置か？ 膨張用のクリアランスはあるか？
- [ ] **内部スペース:** 理想ダイオード変換基板（約10x10mm程度）を収める隙間はあるか？
- [ ] **組立:** 蓋を閉める際、カメラのFPCケーブルを挟み込まない構造か？
